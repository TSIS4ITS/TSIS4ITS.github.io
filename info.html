Primeri pitanja i odgovora

Q1: Sta je css?

A1: CSS (Cascading Style Sheets) je tehnologija koja se koristi zajedno sa HTML-om za definisanje izgleda i rasporeda sadržaja na web stranicama. Omogućava odvajanje strukture (HTML) od prezentacije (stilova), čime se olakšava dizajn i održavanje web stranica.


Q2: Sta je IT?

A2: IT (informacione tehnologije) obuhvata oblasti vezane za upotrebu računara, uključujući hardver, softver, računarske nauke i inženjering, kao i razne profesije i organizacije povezane sa računarima.


Q3: Sta je hardver?

A3: Hardver obuhvata sve fizičke, opipljive delove računara, kao što su elektronski sklopovi, čipovi, grafičke i zvučne kartice, memorija (RAM), matična ploča, monitori, napajanja, kablovi, tastature, štampači i miševi. Ukratko, hardver su svi delovi računara koje možemo fizički dodirnuti i videti.


Q4: Koliko je stara IT industrija?

A4: Na osnovu dostupnih informacija iz Knowledge Base, precizna starost IT industrije nije direktno navedena. Međutim, razvoj programskih jezika i softverskog inženjeringa beleži se kroz više decenija, sa značajnim uticajem različitih tehnologija i jezika tokom vremena.


Q5: Sta je c# jezik?

A5: C# je višenamenski, visokorazinski programski jezik koji podržava više paradigmi, uključujući strukturirano, objektno-orijentisano, funkcionalno i još mnogo toga. Razvio ga je Anders Hejlsberg u Microsoftu, a prvi put je objavljen 2000. godine. Jezik je dizajniran da bude jednostavan, moderan i siguran za razvoj širokog spektra aplikacija, posebno na platformi .NET.


Q6: Koliko je star HTML?

A6: HTML (HyperText Markup Language) je prvi put javno opisan 1991. godine, a zvanično je objavljen 1993. godine. Dakle, HTML je star više od 30 godina.


Q7: Koliko verzija HTMLa postoji?

A7: HTML je imao više zvaničnih verzija tokom svog razvoja: HTML 2.0, HTML 3.2, HTML 4.0, HTML 4.01, HTML5, kao i nekoliko podverzija (npr. HTML 5.1, 5.2). Takođe, postojale su i varijacije kao što su Strict, Transitional i Frameset za određene verzije. Najnoviji standard je HTML Living Standard koji se stalno ažurira.


Q8: Da li mozes da mi kazes kada i ko je napravio prvi kompjuter?

A8: Prvi koncept programabilnog računara osmislio je Čarls Bebidž (Charles Babbage), koji je 1822. godine predstavio svoju ideju mehaničkog računara nazvanog "difference engine". On se smatra "ocem računara" jer je prvi razvio koncept programabilnog računara u 19. veku. Kasnije, tokom 20. veka, razvijeni su prvi elektronski digitalni računari, kao što su Z3 (Konrad Zuse, 1941), Colossus (Velika Britanija, 1943–1944) i ENIAC (SAD, 1945).


Q9: Sta je razlika izmedju C i C#?

A9: C je proceduralni programski jezik razvijen 1970-ih, dok je C# višenamenski, objektno-orijentisani jezik razvijen od strane Microsofta 2000. godine. C# podržava više programskih paradigmi (objektno-orijentisano, funkcionalno, generičko programiranje), ima automatsko upravljanje memorijom (garbage collection) i koristi se uglavnom za razvoj aplikacija na .NET platformi. C je bliži hardveru, nema automatsko upravljanje memorijom i koristi se za razvoj sistema i ugrađenih aplikacija. Sintaksa C# jezika je slična C-u, ali je znatno modernizovana i proširena dodatnim mogućnostima.


Q10: Zar nisu Java i JavaScript ista stvar?

A10: Java i JavaScript nisu ista stvar. Iako imaju slična imena i delimično sličnu sintaksu, to su potpuno različiti programski jezici, razvijeni od strane različitih ljudi i kompanija, sa različitim konceptima i primenama. Java je statički tipiziran, kompajliran jezik, dok je JavaScript dinamički tipiziran i interpretiran jezik. Java koristi klasu kao osnovu za objekte, dok JavaScript koristi prototipove. Sličnost u imenima je rezultat marketinške odluke, a ne tehničke povezanosti.


Q11: Za sta se PHP koristi?

A11: PHP je skriptni jezik opšte namene koji je posebno prilagođen za serversko web programiranje. Najčešće se koristi za izradu dinamičkih web stranica i aplikacija na strani servera, gde omogućava generisanje sadržaja koji se prikazuje korisnicima. PHP se može koristiti i za komandno-linijsko programiranje i razvoj grafičkih aplikacija, ali je njegova glavna primena u web razvoju, kao deo popularnih softverskih paketa kao što su LAMP, WAMP i MAMP. Takođe, koristi se za izradu sistema za upravljanje sadržajem kao što su WordPress, Joomla i Drupal.


Q12: Prvi put cujem za jezik Ruby, za sta se koristi?

A12: Ruby je višenamenski programski jezik koji se koristi za razvoj različitih vrsta softvera, a posebno je poznat po jednostavnosti i produktivnosti. Najčešće se koristi za web razvoj (posebno uz Ruby on Rails framework), ali i za skriptovanje, automatizaciju, razvoj alata i prototipova. Ruby podržava više programskih paradigmi, uključujući objektno-orijentisano, funkcionalno i proceduralno programiranje, i poznat je po fleksibilnoj i čitljivoj sintaksi.


Q13: Da li znas neki softver za pravljenje igrica?

A13: Neki od najpoznatijih softvera za pravljenje igrica su Unreal Engine i Unity. Unreal Engine je 3D grafički game engine razvijen od strane Epic Games i koristi se za razvoj igara na različitim platformama. Unity je cross-platform game engine koji je popularan među indie developerima i koristi se za izradu 2D i 3D igara, kao i simulacija. Takođe, postoje i drugi alati kao što su Godot Engine, GameMaker, RPG Maker i Construct.


Q14: Ko je napravio jezik Carbon?

A14: Programski jezik Carbon je razvijen u okviru kompanije Google. Prvi put ga je javno predstavio inženjer Chandler Carruth na konferenciji CppNorth u julu 2022. godine. Carbon je zamišljen kao eksperimentalni naslednik C++ jezika i projekat je otvorenog koda.


Q15: Da li je Carbon uspeo da nasledi/zameni c++?

A15: Carbon je eksperimentalni programski jezik koji je zamišljen kao naslednik C++, ali još uvek nije uspeo da zameni C++. Prva eksperimentalna verzija (MVP 0.1) se očekuje najranije krajem 2026. godine, dok se stabilna verzija 1.0 planira tek nakon 2028. Dakle, Carbon još nije spreman za široku upotrebu i nije zamenio C++.
